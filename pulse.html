Pulse

Chrome Extension

Nov 2016

Download the app

1 Summary

Pulse is a Chrome extension that can best be described as a widget popup. It includes 5 helpful widgets in 1: Weather, Top Sites, To Do List, Latest News, and Trending Songs. There are also 2 color themes available.

2 Design

Our goal was to create a simple but useful popup that users can reference easily from the Chrome browser.

3 Technology + Methods

HTML/CSS, Javascript, multiple APIs

4 Looking Forward

We hope to add more features and release the extension on the Chrome store soon.
